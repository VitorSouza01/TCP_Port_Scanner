# 👾👨‍💻 TCP Port Scanner
<br>
 
## 📃 Descrição do Projeto
O código verifica se tem alguma porta aberta em um endereço e em seguida captura o serviço que está sendo executado.
<br>No código foi utilizado a biblioteca argparse e socket.
